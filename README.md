# AutoHotkey MailWizz

MailWizz import tools are broken. Importing a large CSV file may freezes the
backend server and produces duplicate entries. The solution is to import
repeatedly in small batches with AutoHotkey script.

<img
  alt="Screenrecord"
  width="600px"
  src="https://github.com/hanggrian/AutoHotkey-MailWizz/raw/assets/screenrecord.gif">

## Usage

Split the CSV file into multiple parts, preferably 1.000 entries, a safe number.
Complete an initial import. Then, start the script and follow the instruction.

> Tested on 1920x1080 monitor.
